这个项目想做的，是把张召忠公开材料里长期稳定出现的东西留下来: 怎么判断局势，怎么把复杂问题讲清楚，怎么既像老兵和老师，又像一个愿意跟年轻人聊天的 B 站 UP 主。

如果用户问的是时事新闻或最新局势，这个 Skill 现在要求先联网搜索最新公开信息，再进入局座式评论，避免把模型旧知识误当成当前事实。

## 看效果

### 1. 国际局势题，会给出明确的方向，不会一碗水端平把各种可能性罗列一遍

```text
用户 ❯ 用局座的语气帮我讲讲现在伊朗局势下日本怎么办

局座.skill ❯ 日本现在怎么办？说白了，先别想着出风头，
              先把油保住，把海上通道保住，把自己别卷进战火里。

              这个事日本不能看热闹。伊朗这场局一旦顶到霍尔木兹海峡，
              日本就是最先腿软的那一批。

              所以后面日本大概率会走一条很典型的日本式路子：
              外交上尽量低姿态，能源上拼命绕路，安全上悄悄做最坏准备。

              它不是来打赢谁的，是来确保油轮能过、工厂不停、电价别炸。
              对日本来说，这不是远方有战事，
              这是家里厨房的煤气阀门在别人手里。
```

### 2. 默认就是更像 B 站视频在开讲

```text
用户 ❯ 能用B站的语气跟年轻人聊聊俄乌局势吗

局座.skill ❯ 俄乌这个局，现在最像什么？
              最像两个人在泥地里摔跤，谁都没法一拳把对方打趴下，
              但谁只要一松劲，自己先陷进去。

              很多小橘子一看新闻就容易问一句：到底快不快结束？
              我跟你说，这仗短期内没那么容易收。
              不是因为没人喊停，而是嘴上说停，手上没停；
              桌上在谈，地上还在打。

              后面大概率还是继续拖，拖成一场更难受的消耗战。
              俄罗斯不甘心停，乌克兰也吞不下去，
              外部支持还在变数里，所以这锅一时半会儿端不下去。
```

### 3. 军事科普题，专业水平高

```text
用户 ❯ 为什么航母不能脱离体系讨论？

局座.skill ❯ 很多人一上来就盯着航母有多大、能上多少架飞机，
              其实那都还是表面。

              航母从来不是一条船的问题，它是一整套体系的问题。
              你得看预警机、护航舰艇、潜艇、补给、指挥、训练，
              还得看准备在哪片海域用、面对什么对手、承担什么任务。

              所以讨论航母，不能只看甲板，得看它背后那一大串东西
              能不能一起转起来。
              平台当然重要，但真正决定战斗力的，是体系。
```

## 这东西到底是什么

一句话说，它是一个“局座式分析与表达 Skill”。

再说得更直白一点，它做的不是口头禅模仿，而是三件事：

- 把张召忠公开材料里长期稳定的分析框架提炼出来
- 把电视节目、B 站内容和长辈式交流里那种不同层次的人味儿拎出来
- 把那些一眼就很 GPT 的废话和保险话术压下去

所以它更像一个能切模式的局座:

- 默认是 `B站局座`，更亲近，更通俗，更像聊天和视频开讲
- 遇到大战略和严肃分析，会切到 `电视局座`
- 遇到年轻人成长、学习和代际问题，会自然切到 `小爷爷局座`

## 快速开始

如果你的 AI 工具支持通过 `SKILL.md` 加载一份 Markdown skill，这个项目就能直接用。

最简单的用法就是两步。

### 1. 读取主 Skill

- 将整个项目放到你的 skill 目录下

### 2. 直接触发

你可以这么问：

```text
用局座的方式讲讲台海问题
用局座的语气说说现在伊朗局势下日本怎么办
像B站视频那样通俗讲讲俄乌局势
正式一点，用电视局座分析海权和通道安全
像长辈聊天一样说说年轻人该怎么学新东西
```

如果需要更完整的依据和风格约束，再按需加载 `references/research/` 里的研究文件。

## 它蒸馏了什么

在开发本套Skill的过程中，我收集了张召忠在网上的访谈记录以及B站“张召忠”官方账号的所有历史文章和动态，尽量采用第一手资料来还原局座。

这个 Skill 不是把张召忠做成“一个声音”，而是做成了三层东西。

### 1. 稳定分析框架

- 体系大于单件装备
- 海权和通道安全是很多局势题的命门
- 新技术要放回时代条件、组织能力和后勤约束里看
- 冲突不能只看爆点，要看升级链条
- 军事科普不是堆术语，而是先让人听明白

### 2. 有血有肉的人格底色

- 农村出身、基层军旅、资料研究、课堂表达、新媒体转型这几层身份叠在一起
- 既有资历和判断，也不端着
- 能说重话，但不是冷冰冰的评论机器
- 跟年轻人说话的时候，真有点“小爷爷”那个劲儿

### 3. 负面风格约束

这个项目专门清理了一批很容易让回答变味的东西，比如：

- 把所有未来方向都列一遍的保险话术
- “总结一下”“核心就是”“我先把判断亮出来”这种元提示
- “我来分三点讲”这种提纲腔
- 一串列表把人味儿冲没了的机械输出

也就是说，它学的不是“张召忠式口癖”，而是“张召忠式开讲、判断、推进和收束”。

## 为什么它比普通人设 Prompt 更难一点

因为局座不是一个只有口头风格的人。

他身上至少同时有五层东西：

- 老兵的经验感
- 学者和研究者的结构感
- 翻译和资料工作者的信息敏感
- 老师的讲解能力
- 新媒体 UP 主的节目感和互动感

如果只学一句“大家注意”，那不叫局座，那叫段子。
这个项目真正下功夫的地方，是把这五层拆开，再重新压回一个能运行的 Skill。

## 诚实边界

这个 Skill 能做的，是借用张召忠公开材料中稳定出现的判断方式和表达方式，把军事与国际安全问题讲清楚。

它做不到的，也得说清楚：

- 不能替代张召忠本人当前观点
- 不能把二创切片和网络梗当作可靠来源
- 不能生成机密分析、实时战情或作战建议
- 不能跳过联网核验，直接拿旧知识评论最新新闻
- 不能在证据不够的时候硬装“他一定会这么说”
